# /cancel-ralph 명령어

활성 Ralph Loop 실행을 취소합니다.

## 사용법
```
/cancel-ralph
```

## 이 명령어의 역할

1. **중단**: 현재 ralph-loop 실행
2. **보존**: 지금까지 완료된 작업
3. **보고**: 진행 상황 요약

## 사용 시점

- 작업 요구사항이 변경됨
- 차단 이슈 발견
- 다른 접근 방식으로 전환 필요
- 긴급 중단 필요

## 취소 트리거

다음 문구에서도 루프가 취소됩니다:
- "루프 중단"
- "ralph 취소"
- "작업 중단"
- "실행 중지"

## 취소 후 출력

```markdown
## Ralph 루프 취소됨

### 진행 상황 요약
- 완료된 작업: 5/8
- 현재 작업: "토큰 검증 구현" (in_progress)
- 남은 작업: 3

### 완료된 작업
1. ✓ 인증 아키텍처 설계
2. ✓ 사용자 모델 및 스키마 생성
3. ✓ JWT 토큰 생성 구현
4. ✓ 로그인 엔드포인트 추가
5. ✓ 로그아웃 엔드포인트 추가

### 대기 중인 작업
6. ○ 토큰 검증 미들웨어 구현
7. ○ 단위 테스트 작성
8. ○ API 문서 추가

### 재개하려면
남은 작업으로 `/ralph-loop`를 사용하거나, 수동으로 완료하세요.
```

## 중요 사항

- 완료된 작업은 보존됨
- 커밋되지 않은 변경사항은 스테이징 상태 유지
- 할 일 목록은 현재 상태 표시
- 새 `/ralph-loop`로 재개 가능

## 관련 명령어

- `/ralph-loop` - 실행 시작 또는 재개
- `/plan` - 남은 작업 재계획
